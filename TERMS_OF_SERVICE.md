## 데이터 수집 및 원격 측정 / Data Collection & Telemetry
본 스크립트는 어떠한 개인정보도 수집하지 않습니다.

스크립트 실행 시, 실행 횟수를 집계하는 목적으로만 원격 서버에 익명 신호가 전송됩니다.

수집되는 정보:
- ✅ 실행 횟수 (익명)

수집되지 않는 정보:
- ❌ 사용자 이름 또는 ID
- ❌ IP 주소
- ❌ 게임 또는 장소 ID
- ❌ 기기 및 시스템 정보
- ❌ 기타 개인 식별 정보 (PII)

수집된 데이터는 통계 목적으로만 사용되며, 특정 개인을 식별하는 데 사용될 수 없습니다.

This script collects no personal information whatsoever.
Upon execution, the script sends a single anonymous signal to a remote server
for the sole purpose of counting the number of executions.

The following data is collected:
- ✅ Execution count (anonymous)

The following data is NOT collected:
- ❌ Username or User ID
- ❌ IP Address
- ❌ Game or Place ID
- ❌ Device or system information
- ❌ Any other personally identifiable information (PII)

This telemetry is used only for statistical purposes and cannot be used
to identify any individual user.
